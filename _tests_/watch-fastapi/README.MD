## Docker 部署
1.安装Docker-compose
2. `git clone https://gitee.com/zy7y/watch-fastapi`
3. `cd watch-fastapi`
4. 执行 `docker-compose up -d`
4. 访问`IP:1339` 后端 `IP:2153`前端
